Politique de confidentialité

Cette politique de confidentialité s'applique au projet libre OpenCitiVibes et à son site de documentation.

Logiciel auto-hébergé

OpenCitiVibes est un logiciel auto-hébergé. Chaque organisation qui déploie OpenCitiVibes est responsable de sa propre politique de confidentialité. Cette page couvre uniquement le site web du projet OpenCitiVibes et le dépôt GitHub.

Pour le site web du projet OpenCitiVibes

Informations que nous collectons

Le site de documentation OpenCitiVibes (ce site) collecte un minimum d'informations :

  • Pas de témoins (cookies) - Nous n'utilisons pas de témoins de suivi
  • Pas d'analytique - Nous n'utilisons pas de services d'analyse tiers
  • Journaux serveur - Les journaux standard du serveur web peuvent enregistrer les adresses IP et les heures d'accès

Services tiers

Ce site web utilise :

  • Google Fonts - Pour la typographie (soumis à la politique de confidentialité de Google)
  • GitHub - Pour l'hébergement du code source et les discussions (soumis à la politique de confidentialité de GitHub)

Pour les organisations qui déploient OpenCitiVibes

Si vous déployez OpenCitiVibes pour votre organisation, vous êtes le responsable du traitement de toutes les données utilisateurs de votre instance. Vous devez :

  1. Créer votre propre politique de confidentialité - Expliquer à vos utilisateurs quelles données vous collectez et comment vous les utilisez
  2. Configurer la conservation des données - Définir des politiques de conservation appropriées
  3. Assurer la conformité - Respecter les lois applicables sur la vie privée (RGPD, CCPA, Loi 25 du Québec, etc.)
  4. Sécuriser votre instance - Utiliser HTTPS, maintenir les logiciels à jour et suivre les meilleures pratiques de sécurité

Données collectées par les instances OpenCitiVibes

Un déploiement typique d'OpenCitiVibes collecte :

Type de données Objectif Conservation
Adresse courriel Authentification du compte, notifications Jusqu'à la suppression du compte
Nom d'affichage Attribution publique des idées/commentaires Jusqu'à la suppression du compte
Idées soumises Fonctionnalité principale de la plateforme Configurable
Commentaires Discussion communautaire Configurable
Votes Priorisation démocratique Configurable
Adresses IP Sécurité, limitation de débit Configurable (par défaut : non stockées)

Fonctionnalités respectueuses de la vie privée

OpenCitiVibes inclut plusieurs fonctionnalités axées sur la protection de la vie privée :

  • Pas de suivi tiers - Pas de traceurs analytiques ou publicitaires dans l'installation par défaut
  • Auto-hébergé - Toutes les données restent sur vos serveurs
  • Suppression de compte - Les utilisateurs peuvent supprimer leur compte et les données associées
  • Collecte minimale de données - Seules les données essentielles sont collectées
  • Navigation anonyme optionnelle - Les utilisateurs peuvent consulter les idées sans compte
  • Notifications configurables - Les utilisateurs contrôlent les notifications qu'ils reçoivent
