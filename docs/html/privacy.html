Privacy Policy

This privacy policy applies to the OpenCitiVibes open source project and documentation website.

Self-Hosted Software

OpenCitiVibes is self-hosted software. Each organization that deploys OpenCitiVibes is responsible for their own privacy policy. This page covers only the OpenCitiVibes project website and GitHub repository.

For the OpenCitiVibes Project Website

Information We Collect

The OpenCitiVibes documentation website (this site) collects minimal information:

  • No cookies - We do not use tracking cookies
  • No analytics - We do not use third-party analytics services
  • Server logs - Standard web server logs may record IP addresses and access times

Third-Party Services

This website uses:

  • Google Fonts - For typography (subject to Google's Privacy Policy)
  • GitHub - For source code hosting and discussions (subject to GitHub's Privacy Policy)

For Organizations Deploying OpenCitiVibes

If you deploy OpenCitiVibes for your organization, you are the data controller for all user data on your instance. You should:

  1. Create your own privacy policy - Explain to your users what data you collect and how you use it
  2. Configure data retention - Set appropriate data retention policies
  3. Ensure compliance - Follow applicable privacy laws (GDPR, CCPA, Quebec Law 25, etc.)
  4. Secure your instance - Use HTTPS, keep software updated, and follow security best practices

Data Collected by OpenCitiVibes Instances

A typical OpenCitiVibes deployment collects:

Data Type Purpose Retention
Email address Account authentication, notifications Until account deletion
Display name Public attribution of ideas/comments Until account deletion
Submitted ideas Core platform functionality Configurable
Comments Community discussion Configurable
Votes Democratic prioritization Configurable
IP addresses Security, rate limiting Configurable (default: not stored)

Privacy-Friendly Features

OpenCitiVibes includes several privacy-focused features:

  • No third-party tracking - No analytics or advertising trackers in the default installation
  • Self-hosted - All data stays on your servers
  • Account deletion - Users can delete their accounts and associated data
  • Minimal data collection - Only essential data is collected
  • Optional anonymous browsing - Users can browse ideas without an account
  • Configurable notifications - Users control what notifications they receive
